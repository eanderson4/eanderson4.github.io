Graphing Toolkit

The data was pulled from MISO's website and loaded into a sqlite3 database. The database is pushed to a csv file that dygraphs can read. Great examples were provided to easily add the needed options to get a beautiful display. In addition, D3 was used and the tutorial from Scott Murray was priceless.

Optimization

CPLEX is a solid solver for linear programs and mixed-integer programs by IBM. CONCERT is an api which has various interfaces for different programming languages. I am using the c++ api to call cplex to solve the power flow problem. CPLEX and CONCERT is available to all students through the Academic Initiative.
